Installation

Nuvolaris can be installed in many environments.

Nuvolaris includes powerful a CLI that manages installation.

Using it can install Nuvolaris on:

  • your your local machine using Docker Desktop

  • a Linux Server accessible with SSH

  • a Kubernetes Cluster accessible with its kubeconfig file.
